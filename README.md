🌤️ Weather App

A beautiful, responsive weather app built with React, Vite, Tailwind CSS, and Framer Motion.
It fetches real-time weather data using the OpenWeatherMap API and displays smooth animations with modern UI transitions.


✨ Features

🔍 Search weather by city name
🌡️ Displays temperature, humidity, pressure, wind, and description
🎨 Dynamic backgrounds that change with weather conditions
💫 Smooth fade and motion effects using Framer Motion
📱 Fully responsive — works on mobile, tablet, and desktop


🧰 Built With

React
Vite
Tailwind CSS
Framer Motion
OpenWeatherMap API
